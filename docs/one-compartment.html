The Awesome Equation
C(t) = Rin⁄kV · (1 − e−kΔt) + Cprev · e−kΔt
Exact solution for a one-compartment model with first-order removal and piecewise-constant input.  •  t1/2 = ln(2) / k
Scenario: A lake (or any environmental reservoir) has reached steady-state concentration from a constant pollutant input. At time zero, the input stops. The model shows exponential decay governed by the first-order removal rate k. You can also model what happens when a new, lower input replaces the old one.
